Women's E-commerce Clothing
Reviews Analysis

Objectives

  • Analyze customer reviews of women’s clothing from an anonymous e-commerce retailer
  • Uncover insights into customer sentiment, product attributes, and purchase behavior
  • Leverage NLP and Machine Learning to extract meaningful patterns from textual reviews
  • Provide actionable recommendations to improve product offerings, enhance personalization, and optimize the overall customer experience

    • Methodology

  • Conducted Exploratory Data Analysis (EDA) to examine rating distributions, recommendation trends, and demographic insights
  • Applied Text Preprocessing: lowercasing, punctuation removal, stopword elimination
  • Performed Text Mining: Word, bigram, and trigram frequency analysis
  • Generated word clouds to visualize common themes
  • Utilized Topic Modeling (LDA) to extract key themes such as fit, material, and design
  • Carried out Sentiment Analysis using TextBlob to classify reviews into positive, neutral, and negative sentiments
  • Built and evaluated classification models (Logistic Regression, Naive Bayes, Random Forest) to predict sentiment
  • Developed a Random Forest recommendation predictor combining TF-IDF features with structured variables (e.g., polarity, subjectivity, word count)

    • Data Overview

    Data Source
    • Kaggle: Bank Customer Churn Prediction

    This dataset contains 23,486 customer reviews from a women’s clothing e-commerce platform, anonymized for privacy. It includes 10 features, including Text-based Data, Numerical Data and Categorical Data.

    Data Dictionary

    Variable Description Type
    Title String variable for the title of the review. Text-based
    Review Text String variable for the review body. Text-based
    Rating(1-5 scale) The product score granted by the customer. Numerical
    Age Positive Integer variable of the reviewers age. Numerical
    Positive Feedback Count Positive Integer documenting the number of other customers who found this review positive. Numerical
    Recommended IND Binary variable stating where the customer recommends the product where 1 is recommended, 0 is not recommended. Categorical
    Division Name Categorical name of the product high level division. Categorical
    Department Name Categorical name of the product department name. Categorical
    Class Name Categorical name of the product class name. Categorical

    Exploratory Data Analysis (EDA)
  • High Satisfaction: 82% of reviews are positive, indicating strong customer approval.
  • Top-performing Items: 57% of clothing items received 100% positive recommendations.
  • Core Audience: Most reviewers are between 35–55 years old, reflecting a middle-aged customer base.
  • Low Feedback Volume: Most reviews have little positive feedback, though some items receive over 100 likes.
  • Strong Correlation: Higher ratings are highly associated with product recommendations (correlation = 0.792).
  • Two Review Patterns: Review lengths show two peaks—around 40 and 100 words—indicating both brief and detailed reviewers.

    1. Text Preprocessing

  • Lowercasing: Convert all text to lowercase for consistency.
  • Removing punctuation: Remove special characters such as “.,!?;:” to maintain textual uniformity.
  • Removing stopwords: Remove commonly used words like “the,” “and,” “is,” which do not contribute significant meaning.

    Predictive Modeling

    1.Feature Engineering

  • Sentiment Features: Extracted polarity and subjectivity scores from reviews using TextBlob.
  • Text Complexity: Calculated word count, sentence count, and average word length to capture review structure.
  • TF-IDF Vectorization: Converted review text into numerical features using the top 5,000 most relevant terms.

    • 2.Model Development

  • Combined TF-IDF text features with structured numerical features (Rating, Polarity, Subjectivity, Word Count, and Average Word Length).
  • This feature set was then used as input for the Random Forest Classifier.
  • The dataset was partitioned into training (80%) and testing (20%) subsets using stratified sampling.
  • The model was trained with 100 estimators and a fixed random seed for reproducibility.

    • #TextBlob
tm["Polarity"] = tm["Review Text"].apply(lambda x: TextBlob(x).sentiment.polarity)
tm["Subjectivity"] = tm["Review Text"].apply(lambda x: TextBlob(x).sentiment.subjectivity)
										
#complexity
tm["Word Count"] = tm["Review Text"].apply(lambda x: len(re.findall(r'\b\w+\b', x)))
tm["Sentence Count"] = tm["Review Text"].apply(lambda x: len(re.split(r'[.!?]', x)))
tm["Avg Word Length"] = tm["Review Text"].apply(lambda x: np.mean([len(word) for word in re.findall(r'\b\w+\b', x)]))
							
#TF-IDF
vectorizer = TfidfVectorizer(max_features=5000)
X_text = vectorizer.fit_transform(tm["Cleaned Review"])
										
#combined features
X_features = np.hstack((X_text.toarray(), tm[["Rating", "Polarity", "Subjectivity", "Word Count", "Avg Word Length"]].values))
y = tm["Recommended IND"]
										
#data
X_train, X_test, y_train, y_test = train_test_split(X_features, y, test_size=0.2, random_state=42)

    Key Takeaways

  • 77.1% of reviews are positive, with size and fit being the most commonly discussed factors.
  • Negative reviews often cite sizing issues, fabric quality, and returns, which are key drivers of dissatisfaction.
  • Customers value comfort and material as shown by frequent positive words like soft, flattering, and well-made.
  • Random Forest model achieved 94% accuracy in predicting recommendations, but had lower recall for non-recommended items, highlighting class imbalance issues.
